# M1: Shared Layer Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or
> superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Restructure the shared layer from deck-centric to submission-centric model, add all port interfaces for M2-M5.

**Architecture:** Hexagonal architecture with domain types at the core, port interfaces in `shared/ports/`, and Drizzle
schema as the persistence layer. All external dependencies are injected via ports; domain code has zero I/O.
`neverthrow` `ResultAsync` for all fallible operations.

**Tech Stack:** Deno, Cloudflare Workers, Drizzle ORM (D1/SQLite), Zod, neverthrow

**Spec:** `docs/superpowers/specs/2026-03-16-m1-shared-layer-design.md`

```typescript
import type { ResultAsync } from "neverthrow";
import { z } from "@zod/zod";
import type { QueueError } from "../domain/errors.ts";

export const GenerateCardMessage = z.object({
  type: z.literal("generate_card"),
  cardId: z.number().int().min(1),
});

export type GenerateCardMessage = z.infer<typeof GenerateCardMessage>;

export const QueueMessage = z.discriminatedUnion("type", [
  GenerateCardMessage,
]);

export type QueueMessage = z.infer<typeof QueueMessage>;

export interface QueuePort {
  send(message: QueueMessage): ResultAsync<void, QueueError>;
}
```
